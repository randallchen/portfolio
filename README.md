# Portfolio project


The purpose of this portfolio project is two-fold
	
	1. Show how to rETL (import) data to Braze
	2. Demonstrate an ability to reliably query from APIs


**Tools**
- Orchestration: Prefect
- Transformation/DQ: dbt
- Read: DuckDB
